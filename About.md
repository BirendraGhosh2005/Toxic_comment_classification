*Small Description for "ToxiGuard"*

*ToxiGuard* is an AI-powered web application for real-time toxic comment classification on social media. 

Using *Llama 3.3 LLM* with *n8n workflows*, it listens to Facebook and Instagram comments via webhooks, analyzes the text for hate speech and sarcasm, and instantly flags toxic content on a *Next.js + Supabase* dashboard. 

The goal: automate moderation, protect online communities, and help admins take action in under 1.5 seconds with 94% accuracy.
